# Employee Management System (CRUD)

This project implements a basic **Employee Management System** in Java using Object-Oriented Programming principles.  
It was developed as part of internship tasks to practice class design, encapsulation, and CRUD operations with in-memory data storage.

---

## Features
- Add new employees (Create)
- View all employees (Read)
- Update employee details (Update)
- Delete employees by ID (Delete)
- Console-based menu interface
- In-memory storage using `ArrayList`

---

## Project Structure
employee-management-system/
